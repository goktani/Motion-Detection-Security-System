# Motion Detection Security System

This project uses **motion detection** and **AI-based image processing** to enhance security through **camera integration**.

### Features:
- Motion detection with OpenCV and HOG algorithm.
- Capture and store images of detected movements.
- Real-time notifications via Telegram Bot.
